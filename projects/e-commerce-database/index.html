An Insight into an E-Commerce Database

Project overview

  • This is a completed team project
  • Created a fully functional e-commerce database that can be used by both consumers and producers
  • Used a number of triggers, indexes and views to simulate a real e-commerce database
  • We built the database using Python and SQLite in DB Browser

For this project, we wanted to mimic a database of a huge e-commerce site like Amazon. Then, we identified how a huge database like Amazon’s could be broken down into operations and insights. And we decided to focus on the insights portion because there is more room to interpret results. So, this database serves both customers and producers with insights into products. Our final product was our query outputs. This is an academic project for ST207.
