Predicting Meta’s stock prices using LSTM and MLP

Project overview

  • This is an ongoing project
  • Forecasted future prices of Meta’s stock using recurrent and deep neural networks (Recurrent neural network, long short-term memory and multilayer perceptron)
  • Created a vanilla RNN, LSTM, and MLP model to compare performance
  • Measured accuracy using root mean-squared error
  • Identified the better performing models and provided reasonings
  • Comparing deep learning methods and traditional time series models (ARIMA)
  • Analysis was carried out on Python (Spyder & Jupyter Notebook & Google Colab)
